# System Tools

## 概述

此模块为OpenWrt系统安装常用的网络诊断和系统管理工具，提供完整的系统管理和故障排查能力。

## 功能

- 网络诊断和测试工具
- 系统管理和维护工具
- 文本编辑和处理工具
- 连接跟踪和监控工具

## 包含的工具

### 网络工具
- **bind-dig** - DNS查询和诊断工具
  - 域名解析测试
  - DNS服务器查询
  - 网络连通性诊断

- **wget** - 命令行下载工具
  - HTTP/HTTPS文件下载
  - 支持断点续传
  - 批量下载功能

- **curl** - 多协议数据传输工具
  - HTTP/HTTPS/FTP等协议支持
  - API接口测试
  - 数据传输和调试

### 网络诊断
- **nping** - 网络数据包生成器
  - 自定义数据包发送
  - 网络延迟测试
  - 端口连通性检测

- **tcpdump** - 网络数据包分析器
  - 实时数据包捕获
  - 网络流量分析
  - 协议调试工具

### 系统工具
- **vim-full** - 功能完整的文本编辑器
  - 语法高亮支持
  - 多种编辑模式
  - 插件扩展功能

- **diffutils** - 文件比较工具集
  - diff - 文件差异比较
  - cmp - 二进制文件比较
  - 配置文件对比

- **conntrack** - 连接跟踪工具
  - 网络连接状态监控
  - NAT连接追踪
  - 防火墙连接管理

## 配置文件

- `packages` - 工具软件包列表

### 网络诊断

## 常用命令示例

```bash
# DNS查询
dig google.com

# 下载文件
wget https://example.com/file.tar.gz

# 端口测试
nping -p 80 target.com

# 数据包捕获
tcpdump -i eth0 host 192.168.1.1

# 连接跟踪
conntrack -L
```

## 注意事项

- 工具集占用一定存储空间
- 根据实际需求选择安装
- 部分工具需要root权限
- 建议了解各工具基本用法
